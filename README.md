Running Locally

Clone the repository:

git clone https://github.com/lfroshan/portfolio.git

Open the project folder:

cd portfolio

Launch index.html in your browser.

⸻

Deploying to GitHub Pages

Option 1: User Site

Repository name:

lfroshan.github.io

GitHub Pages URL:

https://lfroshan.github.io

Option 2: Project Site

Repository name:

portfolio

GitHub Pages URL:

https://lfroshan.github.io/portfolio

Enable GitHub Pages:

1. Open repository Settings
2. Navigate to Pages
3. Under Source select:
   - Deploy from branch
4. Select:
   - Branch: main
   - Folder: / (root)
5. Save

GitHub will automatically publish the website.
